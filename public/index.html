This is a client-side JavaScript implementation of Black-Scholes pricing for demonstration purposes. The full C++20 engine with multi-threaded Monte Carlo simulation requires a local build.

To run the full engine locally:
1. Clone: github.com/YOUR_USERNAME/quant-pricing-engine
2. Build: cmake -B build && cmake --build build --config Release
3. Run: cd python_app && python app.py

Features in full version: Multi-threaded Monte Carlo (1.59ms for 1M paths), Python bindings, 73 automated tests
